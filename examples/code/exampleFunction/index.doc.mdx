## add
`code/exampleFunction`

**AST Node Type:** *FunctionDeclaration*

#### JSDoc annotations:

Adds two numbers together.

```bash
@param {number} a The first number to add.

@param {number} b The second number to add.

@returns {number} The sum of `a` and `b`.
```

}
